# 输入公式

要输入公式，请执行以下步骤。

1.选择一个单元格。

2.要让Excel知道您要输入公式，请键入等号（=）。

3.例如，键入公式A1 + A2。

提示：只需选择单元格A1和单元格A2，而不是键入A1和A2。

4.将单元格A1的值更改为3。

Excel会自动重新计算单元格A3的值。这是Excel最强大的功能之一！

# 编辑公式

选择单元格时，Excel将显示公式栏中单元格的值或公式。

1.要编辑公式，请单击公式栏并更改公式。

2.按Enter键。

# 运营商优先权

Excel使用发生计算的默认顺序。如果公式的一部分在括号中，则首先计算该部分。然后执行乘法或除法计算。完成后，Excel将添加和减去公式的其余部分。请参阅下面的示例。

首先，Excel执行乘法（A1 * A2）。接下来，Excel将单元格A3的值添加到此结果中。

# 插入一个函数

每个功能都具有相同的结构。例如，SUM（A1：A4）。此函数的名称是SUM。括号（参数）之间的部分表示我们为Excel提供范围A1：A4作为输入。此功能可添加单元格A1，A2，A3和A4中的值。要记住每个任务使用哪个函数和哪个参数并不容易。幸运的是，Excel中的“插入函数”功能可以帮助您完成此操作。

要插入函数，请执行以下步骤。

1.选择一个单元格。

2.单击“插入函数”按钮。

将出现“插入功能”对话框。

3.搜索功能或从类别中选择功能。例如，从统计类别中选择COUNTIF。

4.单击“确定”。

将出现“功能参数”对话框。

5.单击“范围”框并选择范围A1：C2。

6.单击Criteria框，然后键入> 5。

7.单击“确定”。

结果。该COUNTIF函数计算是大于5的细胞数量。

注意：只需键入= COUNTIF（A1：C2，“> 5”），而不是使用“插入函数”功能。到达时：= COUNTIF（而不是键入A1：C2，只需选择范围A1：C2。
