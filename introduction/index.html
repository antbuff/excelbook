# 入门

本节适用于不了解Excel的用户。

Microsoft Excel是有史以来最常用的软件应用程序之一。全世界有数亿人使用Microsoft Excel。您可以使用Excel输入各种数据并执行财务，数学或统计计算。

1 范围：Excel中的范围是两个或更多单元格的集合。本章概述了一些非常重要的范围操作。

2 公式和函数：公式是计算单元格值的表达式。函数是预定义的公式，并且已在Excel中可用。
