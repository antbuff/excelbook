# 排列

一维数组 | 二维阵列

一个阵列是一组变量。在Excel VBA中，您可以使用数组名称和索引号引用数组的特定变量（元素）。

# 一维数组

要创建一维数组，请执行以下步骤。

在工作表上放置一个命令按钮并添加以下代码行：

Dim Films(1 To 5) As String

Films(1) = "Lord of the Rings"
Films(2) = "Speed"
Films(3) = "Star Wars"
Films(4) = "The Godfather"
Films(5) = "Pulp Fiction"

MsgBox Films(4)

单击工作表上的命令按钮时的结果：

说明：第一个代码行声明一个名为Films的String数组。该数组由五个元素组成。接下来，我们初始化数组的每个元素。最后，我们使用MsgBox显示第四个元素。

# 二维阵列

要创建二维数组，请执行以下步骤。这次我们将从表格中读取名称。

在工作表上放置一个命令按钮并添加以下代码行：

Dim Films(1 To 5, 1 To 2) As String
Dim i As Integer, j As Integer

For i = 1 To 5
    For j = 1 To 2
        Films(i, j) = Cells(i, j).Value
    Next j
Next i

MsgBox Films(4, 2)

单击工作表上的命令按钮时的结果：

说明：第一个代码行声明一个名为Films的String数组。该阵列有两个维度。它由5行和2列组成。提示：行先行，然后行。其他两个Integer类型的变量用于Double循环来初始化数组的每个元素。最后，我们在第4行和第2列的交叉处显示元素。
