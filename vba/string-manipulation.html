# 字符串操作

加入字符串 | 左 | 对 | 中 | Len | INSTR

在本章中，您将找到在Excel VBA中****操作字符串的最重要的函数。

在工作表上放置一个命令按钮，并在下面添加代码行。要执行代码行，请单击工作表上的命令按钮。

# 加入字符串

我们使用＆运算符来连接（连接）字符串。

码：

Dim text1 As String, text2 As String
text1 = "Hi"
text2 = "Tim"

MsgBox text1 & " " & text2

结果：

注意：要插入空格，请使用“”

# 剩下

要从字符串中提取最左边的字符，请使用Left。

码：

Dim text As String
text = "example text"

MsgBox Left(text, 4)

结果：

# 对

要从字符串中提取最右边的字符，请使用Right。我们也可以直接在函数中插入文本。

码：

MsgBox Right("example text", 2)

结果：

# 中

要从字符串中间开始提取子字符串，请使用Mid。

码：

MsgBox Mid("example text", 9, 2)

结果：

注意：从位置9（t）开始，长度为2.如果要从字符串中间开始提取子字符串，直到字符串结尾，则可以省略第三个参数。

# 莱恩

要获取字符串的长度，请使用Len。

码：

MsgBox Len("example text")

结果：

注意：包括空间（位置8）！

# INSTR

要在字符串中查找子字符串的位置，请使用Instr。

码：

MsgBox Instr("example text", "am")

结果：

注意：字符串“am”位于第3位。
