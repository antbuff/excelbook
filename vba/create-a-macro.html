# 创建一个宏

开发人员标签 | 命令按钮 | 分配宏 | Visual Basic编辑器

使用Excel VBA，您可以通过编写所谓的宏来自动执行Excel中的任务。在本章中，将学习如何创建一个在单击命令按钮后执行的简单宏。首先，打开“开发人员”选项卡。

# Developer Tab

要打开“开发人员”选项卡，请执行以下步骤。

1.右键单击功能区上的任意位置，然后单击“自定义功能区”。

2.在“自定义功能区”下，在对话框的右侧，选择“主选项卡”（如有必要）。

3.选中“开发人员”复选框。

4.单击“确定”。

5.您可以在“视图”选项卡旁边找到“开发人员”选项卡。

# 命令按钮

要在工作表上放置命令按钮，请执行以下步骤。

1.在“ 开发工具”选项卡上，单击“插入

2.在“ActiveX控件”组中，单击“命令按钮”。

3.拖动工作表上的命令按钮。
