# 功能和子

功能 | 子

之间的区别功能和子在Excel中VBA是一个函数可以返回一个值，而子不能。随着程序大小的增加，函数和subs变得非常有用。

# 功能

如果希望Excel VBA执行返回结果的任务，则可以使用函数。将函数放入模块（在Visual Basic编辑器中，单击“插入”，“模块”）。例如，名称为Area的函数。

Function Area(x As Double, y As Double) As Double

Area = x * y

End Function

说明：此函数有两个参数（类型为Double）和一个返回类型（As之后的部分也是Double类型）。您可以在代码中使用函数名称（Area）来指示要返回的结果（此处为x * y）。

现在，您可以通过简单地使用函数的名称并为每个参数赋值来从代码中的其他位置引用此函数（换句话说，调用函数）。

在工作表上放置一个命令按钮并添加以下代码行：

Dim z As Double

z = Area(3, 5) + 2

MsgBox z

说明：该函数返回一个值，因此您必须在代码中“捕获”此值。您可以使用另一个变量（z）。接下来，您可以为此变量添加另一个值（如果需要）。最后，使用MsgBox显示值。

# 子

如果希望Excel VBA执行某些操作，则可以使用sub。将子放入模块（在Visual Basic编辑器中，单击“插入”，“模块”）。例如，名为Area的子。

Sub Area(x As Double, y As Double)

MsgBox x * y

End Sub

说明：此子有两个参数（类型为Double）。它没有返回类型！您可以通过简单地使用sub的名称并为每个参数赋值来从代码中的其他位置引用此sub（调用sub）。

在工作表上放置一个命令按钮并添加以下代码行：

Area 3, 5
