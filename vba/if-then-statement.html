# 如果那么声明

如果是声明 | 其他声明

使用如果然后声明在Excel VBA中，如果满足特定条件时执行的代码行。

# 如果那么声明

在工作表上放置一个命令按钮并添加以下代码行：

Dim score As Integer, result As String
score = Range("A1").Value

If score >= 60 Then result = "pass"

Range("B1").Value = result

说明：如果得分大于或等于60，则Excel VBA返回通过。

注意：如果得分小于60，Excel VBA会将空变量结果的值放入单元格B1中。

# 其他声明

在工作表上放置一个命令按钮并添加以下代码行：

Dim score As Integer, result As String
score = Range("A1").Value

If score >= 60 Then
    result = "pass"
Else
    result = "fail"
End If

Range("B1").Value = result

说明：如果得分大于或等于60，则Excel VBA返回通过，否则Excel VBA返回失败。

单击工作表上的命令按钮时的结果：

注意：只有在Then和没有Else语句之后有一个代码行时，才允许在Then之后直接放置代码行并省略（省略）End If（第一个例子）。否则在单词Then和Else之后开始一个新行，并以End If结束（第二个例子）。
