# 用户窗体

添加控件 | 显示用户形式 | 分配宏 | 测试Userform

本章将教您如何创建Excel VBA Userform。我们要创建的Userform如下所示：

# 添加控件

要将控件添加到Userform，请执行以下步骤。

1.打开Visual Basic编辑器。如果看不到Project Explorer，请单击View，Project Explorer。

2.单击“插入”，“Userform”。如果工具箱未自动出现，请单击“查看”，“工具箱”。您的屏幕应设置如下。

3.添加下表中列出的控件。完成此操作后，结果应与前面显示的Userform图片一致。例如，通过单击“工具箱”中的“文本框”来创建文本框控件。接下来，您可以在Userform上拖动一个文本框。当您到达Car框架时，请记住在将两个选项按钮放入其中之前先绘制此框架。

4.根据下表更改控件的名称和标题。名称在Excel VBA代码中使用。字幕是屏幕上显示的字幕。最好更改控件的名称。这将使您的代码更易于阅读。要更改控件的名称和标题，请单击“视图”，“属性窗口”，然后单击每个控件。

# 显示用户表单

要显示Userform，请在工作表上放置一个命令按钮并添加以下代码行：

Private Sub CommandButton1_Click()

DinnerPlannerUserForm.Show

End Sub

我们现在要创建Sub UserForm_Initialize。当您对Userform使用Show方法时，将自动执行此子程序。

1.打开Visual Basic编辑器。

2.在Project Explorer中，右键单击DinnerPlannerUserForm，然后单击View Code。

3.从左侧下拉列表中选择Userform。从右侧下拉列表中选择“初始化”。

4.添加以下代码行：

Private Sub UserForm_Initialize()

'Empty NameTextBox
NameTextBox.Value = ""

'Empty PhoneTextBox
PhoneTextBox.Value = ""

'Empty CityListBox
CityListBox.Clear

'Fill CityListBox
With CityListBox
    .AddItem "San Francisco"
    .AddItem "Oakland"
    .AddItem "Richmond"
End With

'Empty DinnerComboBox
DinnerComboBox.Clear

'Fill DinnerComboBox
With DinnerComboBox
    .AddItem "Italian"
    .AddItem "Chinese"
    .AddItem "Frites and Meat"
End With

'Uncheck DataCheckBoxes
DateCheckBox1.Value = False
DateCheckBox2.Value = False
DateCheckBox3.Value = False

'Set no car as default
CarOptionButton2.Value = True

'Empty MoneyTextBox
MoneyTextBox.Value = ""

'Set Focus on NameTextBox
NameTextBox.SetFocus

End Sub

说明：清空文本框，填充列表框和组合框，取消选中复选框等。

# 分配宏

我们现在已经创建了Userform的第一部分。虽然它看起来很整洁，但当我们点击Userform上的命令按钮时，什么都不会发生。

1.打开Visual Basic编辑器。

2.在Project Explorer中，双击DinnerPlannerUserForm。

3.双击Money旋转按钮。

4.添加以下代码行：

Private Sub MoneySpinButton_Change()

MoneyTextBox.Text = MoneySpinButton.Value

End Sub

说明：使用旋转按钮时，此代码行会更新文本框。

5.双击“确定”按钮。

6.添加以下代码行：

Private Sub OKButton_Click()

Dim emptyRow As Long

'Make Sheet1 active
Sheet1.Activate

'Determine emptyRow
emptyRow = WorksheetFunction.CountA(Range("A:A")) + 1

'Transfer information
Cells(emptyRow, 1).Value = NameTextBox.Value
Cells(emptyRow, 2).Value = PhoneTextBox.Value
Cells(emptyRow, 3).Value = CityListBox.Value
Cells(emptyRow, 4).Value = DinnerComboBox.Value

If DateCheckBox1.Value = True Then Cells(emptyRow, 5).Value = DateCheckBox1.Caption

If DateCheckBox2.Value = True Then Cells(emptyRow, 5).Value = Cells(emptyRow, 5).Value & " " & DateCheckBox2.Caption

If DateCheckBox3.Value = True Then Cells(emptyRow, 5).Value = Cells(emptyRow, 5).Value & " " & DateCheckBox3.Caption

If CarOptionButton1.Value = True Then
    Cells(emptyRow, 6).Value = "Yes"
Else
    Cells(emptyRow, 6).Value = "No"
End If

Cells(emptyRow, 7).Value = MoneyTextBox.Value

End Sub

说明：首先，我们激活Sheet1。接下来，我们确定emptyRow。变量emptyRow是第一个空行，每次添加记录时都会增加。最后，我们将信息从Userform传输到emptyRow的特定列。

7.双击“清除”按钮。

8.添加以下代码行：

Private Sub ClearButton_Click()

Call UserForm_Initialize

End Sub

说明：单击“清除”按钮时，此代码行将调用Sub UserForm_Initialize。

9.双击“取消”按钮。

10.添加以下代码行：

Private Sub CancelButton_Click()

Unload Me

End Sub

说明：单击“取消”按钮时，此代码行将关闭Userform。
