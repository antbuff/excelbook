# 范围对象

范围示例 | 细胞 | 声明一个范围对象 | 选择 | 行 | 专栏 | 复制/粘贴 | 清除 | 计数

该Range对象，这是你的工作表单元格（或单元）的代表，是最重要的对象Excel的VBA。本章概述了Range对象的属性和方法。属性是对象具有的东西（它们描述对象），而方法做某事（它们对对象执行动作）。

# 范围示例

在工作表上放置一个命令按钮并添加以下代码行：

Range("B3").Value = 2

单击工作表上的命令按钮时的结果：

码：

Range("A1:A4").Value = 5

结果：

码：

Range("A1:A2,B3:C4").Value = 10

结果：

注意：要在Excel VBA代码中引用命名范围，请使用如下代码行：

Range("Prices").Value = 15

# 细胞

您也可以使用单元格代替Range。使用电池的时候要特别有用循环通过范围。

码：

Cells(3, 2).Value = 2

结果：

说明：Excel VBA将值2输入到第3行和第2列交叉处的单元格中。

码：

Range(Cells(1, 1), Cells(4, 1)).Value = 5

结果：

# 声明一个范围对象

您可以使用关键字Dim和Set声明Range对象。

码：

Dim example As Range
Set example = Range("A1:C4")

example.Value = 8

结果：

# 选择

Range对象的一个​​重要方法是Select方法。Select方法只选择一个范围。

码：

Dim example As Range
Set example = Range("A1:C4")

example.Select

结果：

注意：要选择其他工作表上的单元格，必须先激活此工作表。例如，以下代码行从左侧选择第三个工作表上的单元格B7。

Worksheets(3).Activate
Worksheets(3).Range("B7").Select

# 行

“行”属性允许访问范围的特定行。

码：

Dim example As Range
Set example = Range("A1:C4")

example.Rows(3).Select

结果：

注意：边框仅用于说明。

# 列

Columns属性允许访问范围的特定列。

码：

Dim example As Range
Set example = Range("A1:C4")

example.Columns(2).Select

结果：

注意：边框仅用于说明。

# 复制粘贴

复制和粘贴方法用于复制范围并将其粘贴到工作表上的其他位置。

码：

Range("A1:A2").Select
Selection.Copy

Range("C3").Select
ActiveSheet.Paste

结果：

虽然在Excel VBA中允许这样做，但使用下面的代码行完全相同会好得多。

Range("C3:C4").Value = Range("A1:A2").Value

# 明确

要清除Excel范围的内容，可以使用ClearContents方法。

Range("A1").ClearContents

或者只是使用：

Range("A1").Value = ""

注意：使用Clear方法清除范围的内容和格式。使用ClearFormats方法仅清除格式。

# 计数

使用Count属性，您可以计算范围的单元格，行数和列数。

注意：边框仅用于说明。

码：

Dim example As Range
Set example = Range("A1:C4")

MsgBox example.Count

结果：

码：

Dim example As Range
Set example = Range("A1:C4")

MsgBox example.Rows.Count

结果：

注意：以类似的方式，您可以计算范围的列数。
