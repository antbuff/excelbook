# 宏错误

本章将教你如何应对宏观错误在Excel中。首先，让我们创建一些错误。

在工作表上放置一个命令按钮并添加以下代码行：

x = 2
Range("A1").Valu = x

1.单击工作表上的命令按钮。

结果：

2.单击“确定”。

变量x未定义。因为我们在代码开头使用Option Explicit语句，所以我们必须声明所有变量。Excel VBA已将x蓝色着色以指示错误。

3.在Visual Basic编辑器中，单击“重置”以停止调试器。

4.通过在代码开头添加以下代码行来更正错误。

Dim x As Integer

你可能听说过之前调试过的技术。使用此技术，您可以单步执行代码。

5.在Visual Basic编辑器中，将光标置于“专用”之前，然后按F8。

第一行变黄。

6.再按F8三次。

出现以下错误。

Range对象有一个名为Value的属性。这里的值拼写不正确。调试是一种很好的方法，不仅可以找到错误，还可以更好地理解代码。我们的调试示例程序向您展示如何单步执行代码并查看工作表上每个代码行的效果。
