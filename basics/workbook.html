# 工作簿

打开现有工作簿 | 关闭工作簿 | 创建新工作簿 | 关闭“开始”屏幕

一个工作簿是你的另一个词的Excel文件。当您启动Excel时，单击空白工作簿从头开始创建Excel工作簿。

# 打开现有工作簿

要打开过去创建的工作簿，请执行以下步骤。

1.在“文件”选项卡上，单击“打开”。

2.最近显示了最近使用的工作簿列表。您可以从此处快速打开工作簿。

3.单击“浏览”以打开列表中没有的工作簿。

# 关闭工作簿

若要关闭工作簿（和Excel），请单击右上角的X.如果打开了多个工作簿，则单击右上角的X将关闭活动工作簿。

# 创建一个新的工作簿

有时你想重新开始。要创建新工作簿，请执行以下步骤。

1.在“文件”选项卡上，单击“新建”。

2.单击“空白工作簿”。

# 关闭“开始”屏幕

当您启动Excel时，它会显示一个开始屏幕，其中列出了最近使用的Excel文件和模板。要跳过开始屏幕并始终以空白工作簿开头，请执行以下步骤。

1.在“文件”选项卡上，单击“选项”。

2.取消选中“此应用程序启动时显示开始屏幕”。

3.单击“确定”。
