# 打印

打印工作表 | 什么打印 | 多份副本 | 方向 | 页边距 | 缩放

本章将教您如何打印工作表以及如何在Excel中更改一些重要的打印设置。

# 打印工作表

要在Excel中打印工作表，请执行以下步骤。

1.在“文件”选项卡上，单击“打印”。

2.要预览将要打印的其他页面，请单击窗口底部的“下一页”或“上一页”。

3.要打印工作表，请单击“大打印”按钮。

# 打印什么

您也可以只打印当前选择，而不是打印整个工作表。

1.首先，选择要打印的单元格范围。

2.接下来，在“设置”下，选择“打印选择”

3.要打印选择，请单击“大打印”按钮。

注意：您还可以打印活动工作表（首先通过按住CTRL并单击工作表选项卡来选择工作表）或打印整个工作簿。使用Pages旁边的框（请参见第一个屏幕截图）仅打印文档的几页。例如，2到2仅打印第二页。

# 多份副本

要打印多份副本，请执行以下步骤。

1.使用“份数”框旁边的箭头。

2.如果一个副本包含多个页面，则可以在Collat​​ed和Uncollat​​ed之间切换。例如，如果您打印6份，则Collat​​ed打印整个第一份副本，然后打印整份第二份副本等。未分页打印第1页的6份副本，第2份的6份副本等。

# 取向

您可以在纵向方向（更多行但更少列）和横向方向（更多列但更少行）之间切换。

# 页边距

要调整页边距，请执行以下步骤。

1.从“边距”下拉列表中选择一个预定义边距（“正常”，“宽”或“窄”）。

2.或单击窗口右下角的“显示边距”图标。现在，您可以拖动线条以手动更改页边距。

# 缩放

如果要在一个页面上放置更多数据，可以将工作表放在一个页面上。要实现此目的，请执行以下步骤。

1.从Scaling下拉列表中选择“Fit Sheet on One Page”。

注意：您还可以将打印输出缩小到一页宽或一页高。单击“自定义缩放选项”以手动输入缩放百分比或使打印输出适合特定页数的宽度和高度。请注意，当您的打印输出变得不可读时，Excel不会发出警告。
