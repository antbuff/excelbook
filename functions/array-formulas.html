# 数组公式

没有数组公式 | 使用数组公式 | F9键

本章将帮助您了解数组公式在Excel中。单细胞阵列公式在一个细胞中执行多次计算。

# 没有数组公式

不使用数组公式，我们将执行以下步骤以找到最大的进展。

1.首先，我们将计算每个学生的进度。

2.接下来，我们将使用MAX函数来找到最大的进展。

# 使用数组公式

我们不需要在D列中存储范围.Excel可以将此范围存储在其内存中。存储在Excel内存中的范围称为数组常量。

1.我们已经知道，我们可以使用下面的公式找到第一个学生的进度。

2.为了找到最大的进步（不要被淹没），我们添加MAX功能，将C2替换为C2：C6和B2替换为B2：B6。

3.按CTRL + SHIFT + ENTER完成。

注意：公式栏通过将其括在花括号{}中来表示这是一个数组公式。不要自己打字。编辑公式时，它们将消失。

说明：范围（数组常量）存储在Excel的内存中，而不是存储在范围内。数组常量如下所示：

{19; 33; 63; 48; 13}

该数组常量用作MAX函数的参数，结果为63。

# F9键

使用数组公式时，您可以自己查看这些数组常量。

1.在公式中选择C2：C6-B2：B6。

2.按F9。

看起来很好。垂直数组常量中的元素由分号分隔。水平数组常量中的元素用逗号分隔。
