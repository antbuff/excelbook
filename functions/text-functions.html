# 文字功能

加入字符串 | 左 | 对 | 中 | Len | 找到 | 替代

在操作文本字符串时，Excel提供了许多功能。

# 加入字符串

要连接字符串，请使用＆运算符。

注意：要插入空格，请使用“”

# 剩下

要从字符串中提取最左边的字符，请使用LEFT函数。

# 对

要从字符串中提取最右边的字符，请使用RIGHT函数。

# 中

要从字符串中间开始提取子字符串，请使用MID函数。

注意：从位置5（p）开始，长度为3。

# 莱恩

要获取字符串的长度，请使用LEN函数。

注意：包括空间（位置8）！

# 找

要在字符串中查找子字符串的位置，请使用FIND函数。

注意：字符串“am”位于第3位。

# 替代

要使用字符串中的新文本替换现有文本，请使用SUBSTITUTE函数。
