# 统计功能

平均值 | Averageif | 中位数 | 模式 | 标准差 | 最小 | 最大 | 大 | 小

本章提供了一些非常有用的概述统计功能在Excel中。

# 平均

要计算一系列单元格的平均值，请使用AVERAGE函数。

# AVERAGEIF

要根据一个标准对细胞进行平均，请使用AVERAGEIF功能。例如，计算不包括零的平均值。

注意：<>表示不等于。AVERAGEIF功能类似于SUMIF功能。

# 模式

要查找最常出现的数字，请使用MODE功能。

# 标准偏差

要计算标准偏差，请使用STEDV功能。

注意：标准差是一个数字，告诉您数字与平均值的距离。在我们的标准偏差页面上了解有关此主题的更多信息。

# 敏

要查找最小值，请使用MIN功能。

# 马克斯
