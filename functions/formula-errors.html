# 公式错误

#####错误 | ＃名称？错误 | ＃值！错误 | ＃DIV / 0！错误 | ＃REF！错误

本章将教您如何在Excel中处理一些常见的公式错误。

# #####错误

当您的单元格包含此错误代码时，该列的宽度不足以显示该值。

1.单击A列标题的右边框并增加列宽。

提示：双击列A标题的右边框以自动适合A列中最宽的条目。

# ＃名称？错误

名字？Excel无法识别公式中的文本时发生错误。

1.简单地将SU校正为SUM。

# ＃值！错误

Excel显示#VALUE！公式具有错误类型的参数时出错。

1A。将单元格A3的值更改为数字。
1B。使用函数忽略包含文本的单元格。

# ＃DIV / 0！错误

Excel显示#DIV / 0！公式尝试将数字除以0或空单元格时出错。

1A。将单元格A2的值更改为不等于0的值
。使用逻辑功能IF防止显示错误。

说明：如果单元格A2等于0，则显示空字符串（“”）。如果不是，则显示公式A1 / A2的结果。

# ＃REF！错误

Excel显示#REF！公式引用无效的单元格时出错。

1.单元C1引用单元格A1和单元格B1。

2.删除列B.要实现此目的，请右键单击列B标题，然后单击“删除”。

3.选择单元格B1。对单元格B1的引用不再有效。

4.要修复此错误，您可以删除+＃REF！在单元格B1的公式中，您可以通过按CTRL + z撤消操作
