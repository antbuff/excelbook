# 计数和求和函数

数 | Countif | Countifs | 总和 | Sumif | SUMIFS

Excel中最常用的功能是计数和求和的函数。您可以根据一个标准或多个标准对其进行计数和求和。

# 计数

要计算包含数字的单元格数，请使用COUNT函数。

# COUNTIF

要根据一个条件（例如，大于9）计算单元格，请使用以下COUNTIF函数。

注意：有关更多示例，请访问我们关于COUNTIF函数的页面。

# COUNTIFS

要根据多个条件（例如，绿色和大于9）对单元格进行计数，请使用以下COUNTIFS函数。

# 和

要对一系列单元格求和，请使用SUM函数。

# SUMIF

要根据一个条件（例如，大于9）对单元格求和，请使用以下SUMIF函数（两个参数）。

要根据一个条件（例如，绿色）对单元格求和，请使用以下SUMIF函数（三个参数，最后一个参数是要求和的范围）。

注意：有关更多示例，请访问我们关于SUMIF函数的页面。

# SUMIFS

要根据多个条件（例如，圆和红色）对单元格求和，请使用以下SUMIFS函数（第一个参数是要求和的范围）。

一般说明：以类似的方式，您可以使用AVERAGEIF和AVERAGEIFS功能根据一个或多个标准对细胞进行平均。
